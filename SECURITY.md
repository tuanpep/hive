# Security Policy

## Reporting a Vulnerability

We take the security of HIVE-Swarm seriously. If you believe you have found a security vulnerability, please do NOT create a public issue. 

Instead, please report it privately to the maintainers at [tuanbt-security@example.com](mailto:tuanbt-security@example.com).

## Supported Versions

Typically, only the latest version of HIVE-Swarm receives security updates.

## Guidelines

- Provide a detailed description of the vulnerability.
- Include steps to reproduce the issue.
- Provide a proof-of-concept if possible.
- Give us a reasonable amount of time to fix the issue before disclosure.
